# Pencatatan Keuangan

Aplikasi pencatatan keuangan sederhana yang tidak memerlukan instalasi tambahan. Semua modul yang digunakan berasal dari pustaka standar Python.

## Menjalankan aplikasi

1. Pastikan Python 3 sudah terpasang pada sistem.
2. Jalankan salah satu skrip berikut:
   - Linux/macOS: `./start.sh`
   - Windows: `start.bat`

Data transaksi akan disimpan pada berkas `transaksi.csv` di direktori yang sama dengan aplikasi.
